# Configuration
Install required modules and build scraper:
- Linux:
```
go install && go build
```
- Windows (PowerShell):
```
go install ; go build
```
# Usage
- **Linux**: `./iptvcat-scraper channel_name`
- **Windows**: `.\iptvcat-scraper.exe channel_name`

Scraped data will be saved to `all-streams.json`.
